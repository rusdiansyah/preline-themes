## Tentang Aplikasi

    Dibuat menggunakan Laravel 12 , Livewire 4, Tailwindcss & Preline.

### Cara Install

1. clone Aplikasi:
```bash
git clone git@github.com:rusdiansyah/preline-themes.git nama_aplikasi
```
```bash
cd nama_aplikasi
```
```bash
composer update
```
```bash
npm install
```
```bash
npm run dev
```

2. buat file .env
```bash
cp .env.example .env
```

3. edit file .env terutama pada koneksi database

4. migrate table
```bash
php artisan migrate
```

5. buat link storege
```bash
php artisan storage:link
```
